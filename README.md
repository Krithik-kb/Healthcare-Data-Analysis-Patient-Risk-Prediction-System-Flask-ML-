# 🏥 Healthcare Data Analysis & Patient Risk Prediction System

A Flask-based Machine Learning web application that analyzes healthcare data and predicts patient risk levels.

## 🚀 Features
- Patient risk prediction using Machine Learning
- Healthcare data analysis
- Interactive Flask web interface
- Data visualization and insights
- User-friendly dashboard

## 🛠️ Technologies Used
- Python
- Flask
- Machine Learning
- Pandas
- NumPy
- Scikit-learn
- HTML
- CSS
- JavaScript

## 📂 Project Structure
```bash
app.py
templates/
static/
model/
dataset/
```

## ▶️ How to Run

1. Clone the repository
```bash
git clone https://github.com/Krithik-kb/Healthcare-Data-Analysis-Patient-Risk-Prediction-System-Flask-ML-.git
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Run the Flask app
```bash
python app.py
```

4. Open browser
```bash
http://127.0.0.1:5000
```
